# 🚀 Quick Start Guide

Panduan cepat untuk langsung memulai menggunakan website portfolio ini dalam 5 menit!

## ⚡ Langkah Cepat (5 Menit)

### 1️⃣ Download & Extract (30 detik)
```
1. Download file ZIP
2. Extract ke folder pilihan
3. Done!
```

### 2️⃣ Ganti Informasi Pribadi (2 menit)

**File: `index.html`**

Cari dan ganti:
- `Eki Zulfar Rachman` → Nama Anda
- `ekizulfarrachman@gmail.com` → Email Anda
- Link social media (GitHub, LinkedIn, Instagram)

### 3️⃣ Ganti Foto (1 menit)

1. Foto profil: `images/profile.jpg`
2. Hero image: `images/animation.gif`

### 4️⃣ Update Proyek (1 menit)

**File: `js/data.js`**

Edit array `projects` - ganti title, description, tags sesuai proyek Anda.

### 5️⃣ Buka di Browser (30 detik)

```
Klik kanan index.html → Open With → Browser
```

## ✅ Selesai!

Website sudah siap digunakan!

## 🎯 Langkah Selanjutnya

### Kustomisasi Lebih Lanjut
1. Baca `CUSTOMIZATION.md` untuk detail kustomisasi
2. Ganti warna theme di `css/style.css`
3. Tambah lebih banyak proyek

### Deploy Online
1. Baca `README.md` bagian Deploy
2. Pilih hosting (GitHub Pages recommended)
3. Upload dan publish

### Optimize
1. Ganti semua gambar placeholder
2. Compress images dengan TinyPNG
3. Update SEO meta tags
